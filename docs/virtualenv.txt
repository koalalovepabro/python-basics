===== python Isolation Tools(Virtual Environments) ====

1. venv: python3.3부터 기본으로 제공
2. virtualenv: Python2부터 사용해오고 있는 가상환경 라이브러리
3. pyenv: Python Version Manager에 내장된 가상환경 플러그인 (버전매니저를 설치해야 함)
4. conda: Anaconda Python을 설치했을 때 사용할 수 있는 가상환경 모듈

==== venv로 Virtual 환경 만들기

1) 프로젝트 생성
    # mkdir myproj
    # cd myproj

2) 가상환경 생성
 [myproj] # python -m venv venv

3) 가상환경 구동
 [myproj] # source venv/bin/activate (unix 계열)
 [myproj] # venv\Scripts\activate (Windows 계열)

4) 가상환경 빠져나오기
 [myproj] # deactivate
